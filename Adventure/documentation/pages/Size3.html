Size3

Size3 is a three-dimensional size consiting of width, height, and depth components. It is a C++ object represented in Lua via a userdata and metatables. It supports the built-in Lua opertations: +, -, *, /, and # with it other Size3s.

Constructors

  • Size3() - Returns a new Size3 of zero (width = height = depth= 0).
  • Size3( Size3 ) - Returns a new Size3 initialized to the supplied Size3's components.
  • Size3( width, height, depth ) - Returns a new Size3 initialized to the supplied width, height and z parameters.

Instance functions

  • number size3:width() - Returns the width component of the Size3.
  • number size3:width( value ) - Sets the width component of the Size3, then returns the new value.
  • number size3:height() - Returns the height component of the Size3.
  • number size3:height( value ) - Sets the height component of the Size3, then returns the new value.
  • number size3:depth() - Returns the depth component of the Size3.
  • number size3:depth( value ) - Sets the depth component of the Size3, then returns the new value.
