V3

V3 is a three dimensional vector consiting of x, y, and z components. It is a C++ object represented in Lua via a userdata and metatables. It supports the built-in Lua opertations: +, -, *, /, and # with it other V3s.

Constructors

  • V3( V3 ) - Returns a new V3 initialized to the supplied V3's components.
  • V3( x, y, z ) - Returns a new V3 initialized to the supplied x, y and z parameters.
  • V3() - Returns a new V3 of zero (x = y = x= 0).
  • V3Cross( V3, V3 ) - Returns a new V3, which is the result of the cross product of two V3s.
  • V3Lerp( V3, V3, delta ) - Returns a new V3, which is the result of the linear interpolation between two V3s.

Instance functions

  • number v3:x() - Returns the x component of the V3.
  • number v3:x( value ) - Sets the x component of the V3, then returns the new value.
  • number v3:y() - Returns the y component of the V3.
  • number v3:y( value ) - Sets the y component of the V3, then returns the new value.
  • number v3:z() - Returns the z component of the V3.
  • number v3:z( value ) - Sets the z component of the V3, then returns the new value.
  • number v3:Normalize() - Returns a normalized V3.
  • number v3:DistanceTo( V3 ) - Returns the distance between two V3s.
  • number v3:Dot( V3 ) - Returns the dot product of two V3s
  • number v3:Angle( V3 ) - Returns the angle between two V3s in degrees.
  • v3:Inverse() - Returns the inverse of a V3.
