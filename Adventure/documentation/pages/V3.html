V3

V3 is a three dimensional vector consiting of x, y, and z components. It is a C++ object represented in Lua via a userdata and metatables. It supports the built-in Lua opertations: +, -, *, /, and # with it other V3s.

Non-Instanced functions

  • V3( x, y, z ) - Returns a new V3 initialized to the supplirted x, y and z parameters.
  • V3() - Returns a new V3 of zero (x = y = x= 0).
  • V3Cross( v3, v3 ) - Returns a new V3, which is the result of the cross product of two V3s.
  • V3Lerp( v3, v3, delta ) - Returns a new V3, which is the result of the linear interpolation between two v3s.

Instanced functions

  • v3.x() - Returns the x component of the V3.
  • v3.x( value ) - Sets the x component of the V3, then returns the new value.
  • v3.y() - Returns the y component of the V3.
  • v3.y( value ) - Sets the y component of the V3, then returns the new value.
  • v3.z() - Returns the z component of the V3.
  • v3.z( value ) - Sets the z component of the V3, then returns the new value.
  • v3.Normalize() - Returns a normalized v3.
  • v3.DistanceTo( v3 ) - Returns the distance between two v3s.
  • v3.Dot( v3 ) - Returns the dot product of two v3s
  • v3.Inverse() - Returns the inverse of a v3.
