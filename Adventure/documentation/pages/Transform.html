Transform

Transform lets us modify an Object's transformation. A transformation explains an Object's place in the world, and rotation.

Constructors

None - Get the object transform directly from the Object .

Instanced functions

  • transform:SetPosition( V3 ) - Set's the absolute position of the object.
  • V3 transform:GetPosition() - Returns the absolute position of the object.
  • transform:LookAt( V3 ) - Makes the object "look at" a specific position.
  • transform:Orbit( V3, V2, distance ) - Orbits, or rotates an object around a point in space's X and Y axis, at a distance (Z).
  • transform:RotateAbout( V3, rotation ) - Rotates the object about it's x, y, and/or z axis, by rotation.
  • transform:SetRotation( Quaternion ) - Sets the absolute rotation of an object.
  • Quaternion transform:GetRotation() - Returns the absolute rotation of an object.
  • Matrix transform:GetMatrix() - Returns the matrix of an object.
  • transform:PreMul( Quaternion or Matrix ) - Performs a multiply against the object (result = Input * object).
  • transform:PostMul(Quaternion or Matrix ) - Performs a multiply against the object (result = object * Input).
  • transform:SetModelMatrix( Matrix ) - Sets the model matrix of the object. This is a way to make the geometry of the object look different.
  • Matrix transform:GetModelMatrix() - Retuns the model matrix of the object.
