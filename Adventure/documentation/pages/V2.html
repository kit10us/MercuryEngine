V2

V2 is a two dimensional vector consiting of x and components. It is a C++ object represented in Lua via a userdata and metatables. It supports the built-in Lua opertations: +, -, *, /, and # with it other V2s.

Constructors

  • V2( V2 ) - Returns a new V2 initialized to the supplied V2's components.
  • V2( x, y ) - Returns a new V2 initialized to the supplied x and z parameters.
  • V2() - Returns a new V2 of zero (x = y = 0).
  • V2Lerp( V2, V2, delta ) - Returns a new V2, which is the result of the linear interpolation between two V3s.

Instance functions

  • number v2:x() - Returns the x component of the V2.
  • number v2:x( value ) - Sets the x component of the V2, then returns the new value.
  • number v2:y() - Returns the y component of the V2.
  • number v2:y( value ) - Sets the y component of the V2, then returns the new value.
  • number v2:Normalize() - Returns a normalized V2.
  • number v2:DistanceTo( V2 ) - Returns the distance between two V2s.
  • number v2:Dot( V2 ) - Returns the dot product of two V2s
  • number v2:Angle( V2 ) - Returns the angle between two V2s in degrees.
  • v2:Inverse() - Returns the inverse of a V2.
