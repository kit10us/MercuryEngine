TexArea

 [back to top]

Four dimensional texture coordinates consiting of upper-left (ul) TexCoords (ulu, ulv), and lower-right (dr) TexCoords (dru, drv)

Non-Instanced functions

  • TexArea() - Creates a texture area that covers the entire face of an image ( { 0, 0 }, { 1, 1 } ).
  • TexArea( TexArea ) - Create a TexArea from another TexArea.
  • TexArea( TexCoords, TexCoords ) - Create a TexArea from two TexCoords.
  • TexArea( ulu, ulv, dru, drv ) - Create a TexArea from individual components.

Instanced functions

  • number texarea:ulu() - Returns the ulu component of the texarea.
  • number texarea:ulu( value ) - Sets the ulv component of the texarea, then returns the new value.
  • number texarea:ulv() - Returns the ulu component of the texarea.
  • number texarea:ulv( value ) - Sets the ulv component of the texarea, then returns the new value.
  • number texarea:dru() - Returns the dru component of the texarea.
  • number texarea:dru( value ) - Sets the drv component of the texarea, then returns the new value.
  • number texarea:drv() - Returns the dru component of the texarea.
  • number texarea:drv( value ) - Sets the drv component of the texarea, then returns the new value.
  • number texarea:ul() - Create TexCoords from the ul of the texarea.
  • number texarea:ul( ulu, ulv ) - Sets the ulu and ulv components of the texarea, then returns TexCoords from those new values.
  • number texarea:ul( TexCoords ) - Sets the ulu and ulv components of the texarea, then returns TexCoords from those new values.
  • number texarea:dr() - Create TexCoords from the dr of the texarea.
  • number texarea:dr( dru, drv ) - Sets the dru and drv components of the texarea, then returns TexCoords from those new values.
  • number texarea:dr( TexCoords ) - Sets the dru and drv components of the texarea, then returns TexCoords from those new values.
