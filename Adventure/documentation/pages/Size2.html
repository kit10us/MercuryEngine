Size2

Size2 is a two-dimensional size consiting of width, and depth components. It is a C++ object represented in Lua via a userdata and metatables. It supports the built-in Lua opertations: +, -, *, /, and # with it other Size2s.

Constructors

  • Size2( Size2 ) - Returns a new Size2 initialized to the supplied Size2's components.
  • Size2( width, height ) - Returns a new Size2 initialized to the supplied width and height parameters.
  • Size2() - Returns a new Size2 of zero (width = height = 0).

Instance functions

  • number size2:width() - Returns the width component of the Size2.
  • number size2:width( value ) - Sets the width component of the Size2, then returns the new value.
  • number size2:height() - Returns the height component of the Size2.
  • number size2:height( value ) - Sets the height component of the Size2, then returns the new value.
