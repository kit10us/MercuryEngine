Color

Color is used to produce and work with colors. Each component is a value between 0 (lowest) to 1, highest. It is a C++ object represented in Lua via a userdata and metatables. It supports the built-in Lua opertations: +, -, *, /, and # with it other Colors.

Constructors

  • RGBA( red, green, blue, alpha ) - Creates a color.
  • ARGB( alpha, red, green, blue ) - Creates a color.
  • RGB( red, green, blue ) - Creates a color.
  • White( [alpha] ) - Creates a color.
  • Red( intensity [, alpha] ) - Creates a color.
  • Green( intensity [, alpha] ) - Creates a color.
  • Blue( intensity [, alpha] ) - Creates a color.
  • Grey( intensity [, alpha] ) - Creates a color.
  • Black( [alpha] ) - Creates a color.
  • ColorLerp( color, color, delta ) - Returns a color linearly interpreted between two colors. Delta is a value between 0 and 1, where 0 would be entirely the first color, and 1 the second color, and 0.5 a 50/50 mix of the two.

Instance functions

  • number color:r() - Returns the r component of the color.
  • number color:r( value ) - Sets the r component of the color, then returns the new value.
  • number color:g() - Returns the g component of the color.
  • number color:g( value ) - Sets the g component of the color, then returns the new value.
  • number color:b() - Returns the b component of the color.
  • number color:b( value ) - Sets the b component of the color, then returns the new value.
  • number color:a() - Returns the a component of the color.
  • number color:a( value ) - Sets the a component of the color, then returns the new value.
