TexCoords

 [back to top]

Two dimensional texture coordinates consiting of a u and v component. Similar to a V2, however, it's x and y -like components are, instead, u and v. This is to help prevent confusion when we talk of vertex components, so that we can as a vertex's x or a vertex's u value, and know when we are talking about position in 3d or 2d space, or when we are talking about the vertex's texture mapping coordinates.

Non-Instanced functions

  • TexCoords( u, v ) - Returns a new texture coordinate of u and v.
  • TexCoords() - Returns texture coordinates of zero (u = v = 0).

Instanced functions

  • number texcoord:u() - Returns the u component of the texcoord.
  • number texcoord:u( value ) - Sets the v component of the texcoord, then returns the new value.
  • number texcoord:v() - Returns the u component of the texcoord.
  • number texcoord:v( value ) - Sets the v component of the texcoord, then returns the new value.
