Matrix

Matrix is used to create and work with matrices.

Constructors

  • Matrix() - Create a new identity matrix.
  • Matrix( Matrix ) - Create a new matrix that's a copy of another matrix.
  • MatrixZero() - Create a new matrix full of 0s.
  • MatrixTranslate( v3 ) - Create a new translation matrix.
  • MatrixOrthoOffCenterLH( left, right, bottom, top, znear, zfar ) - Create a new orthographic off-centered left-handed matrix.
  • MatrixScale( number ) - Create a new scaling matrix.
  • MatrixScale( v3 ) - Create a new scaling matrix.
  • MatrixRotationAboutAxis( v3, angle ) - Create a new rotation matrix.
  • MatrixPerspectiveFovLH( width, height, znear, zfar ) - Create a new perspective field-of-view left-handed matrix.
  • MatrixRotationX( angle ) - Create a new rotation about the x axis matrix.
  • MatrixRotationY( angle ) - Create a new rotation about the x axis matrix.
  • MatrixRotationZ( angle ) - Create a new rotation about the x axis matrix.
  • MatrixLookAtLH( eye, at, up ) - Create a new "lookat" matrix, where eye is the position of the camera/viewer, at is where to look at, and up is the direction for up; all are v3s.
